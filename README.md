## Instalasi

1. Pastikan Java Development Kit (JDK) telah terpasang di komputer Anda.
2. Clone repo ini:
   ```bash
   git clone https://github.com/Furohito/Java_OOP.git
   ```
3. Buka proyek menggunakan IDE favorit Anda (IntelliJ, VSCode, Eclipse, dsb).
